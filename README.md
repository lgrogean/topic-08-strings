# Topic 08: Strings, Lists, and Dictionaries

**Week 4 | CIS Intro to Programming**

---

## Overview

This week you will work with Python's core data structures. You have full creative freedom on what your program does — pick something that interests you and build it using at least two of the three data structures covered this topic.

## Assignment Prompt

Write a Python program that demonstrates your understanding of strings, lists, and/or dictionaries. Your program should:

- Use at least two of the three data structures (strings, lists, dictionaries)
- Perform at least one meaningful operation on each structure you use (e.g., string formatting, list iteration, dictionary lookup)
- Have a clear purpose — it should be something you could describe to a classmate in one sentence

Some ideas to get you started: a simple contact book, a word frequency counter, a quiz that stores questions and answers in a dictionary, a playlist organizer, or a grocery list manager. These are just suggestions — build whatever makes sense to you.

## Peer Review Prompt

When reviewing a classmate's code this week, consider:

- Did they choose data structures that fit their problem well?
- Is there a place where a different structure (e.g., a dictionary instead of two parallel lists) might have made the code cleaner or easier to read?
- What does their program do, and is it clear from reading the code alone?
